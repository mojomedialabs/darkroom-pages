Scanners and Adapters

DARKROOM Assembly Edition and Professional Edition (versions 8.8 and above) support both serial port and USB bar code scanners.

You can use almost any serial port bar code scanner as long at it reads Code 3 of 9 and has a built-in decoder. If your computer or laptop does not have a serial port (but does have USB), you can purchase a serial-to-USB port adapter.

Any USB bar code scanner which can emulate a keyboard and allows programming to specify prefix/suffix characters can be used. For USB barcode scanners only, you will need to program the scanner to add a prefix character ASCII 02H (STX), and to add a suffix character ASCII 0DH (CR).
