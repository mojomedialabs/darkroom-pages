I have a MAC. Can I use Darkroom?

Darkroom was built for a Windows environment. To run Darkroom on your Mac, you'll have to install Windows. You can run Windows "virtually" using Parallels software (approximately $80) or with VM Ware Fusion (also approximately $80 and they offer a 30-day free trial). You can also use Apple Boot Camp to run Windows non-virtually on your Intel-based Mac.

I have Windows 7. Will Darkroom function properly?

Yes. If you have an older version of Darkroom, you may need to update your Dongle / Key drivers here. Older versions of Darkroom work better if you turn off Windows UAC (user access control). Darkroom Version 9.1 has been tested and evaluated to work without issue with Windows 7.

I am using Darkroom on a 64-bit PC and I am getting an error.

Older security keys, also known as “dongles,” that come with your Darkroom purchase were originally built for 32-bit computers. There is a solution for this! All you need to do is download an update for this driver.

Click here to go to the security key’s manufacturer website, Safenet. At the top of the page you will see a sentence under the header Sentinal Drivers that says, “Click here for the latest Sentinel Driver.” Click that link, allow the download and you should not have any further issue.

Does Darkroom support my camera and/or printer?

We keep an elaborate list of supported cameras and printers that can be found here. As you will see, Darkroom has been built with the most relevant and popular professional equipment in mind. Furthermore, Darkroom will import the following file formats, so rest assured, you’re covered:

  • All standard image file formats like .tif, .bmp, .jpg, .tga, .psd, and .png
  • .mif (MegaVision raw files)
  • .crw (Canon raw format, also .raw or .tif)
  • .nef (Nikon raw format)
  • .dcr (Kodak professional file format)
  • .pcd (Kodak PhotoCD)
  • .pcx (Zsoft)
  • .fpx (Flashpix)
  • .tiff (Kodak proprietary tif)

Do I need to purchase camera or printer software too?

Nope. The beauty of Darkroom is that it saves you money because you will not have to purchase supporting software and drivers from your printer and camera manufacturers. If your hardware is listed here and you own Darkroom you will not need to buy anything else -- if you so choose. (With one exception, many camera manufacturers now require you purchase additional hardware to drop images directly into a computer's hot folder or to use WiFi capabilities. If your camera is listed on our supported hardware page you do not need additional hardware; but if your camera is not listed you may need to see if the camera manufacterer supplies hot folder capability. If it does, you're in luck, with that additional equipment or software, your Darkroom software will be able to collect your images wirelessly just as if it was on our list. We're pretty excited about this, by the way.)

What is the security key? Why can't I download a Darkroom purchase right now?

Darkroom requires a software protection key in order to function. Darkroom will load Evaluation Mode if the software key is not connected properly (in a USB slot). Evaluation Mode does not permit printing and will watermark the images you import into the software. When you purchase Darkroom directly from us or from an authorized dealer, the latest version of the software and the key (also called a “dongle”) will be shipped to you. If you are in a rush, know that most dealers offer overnight shipping.

Whatever happened to Web edition?

Web edition was built as part of an agreement with PhotoReflect.com to develop a robust online publisher that also had some workflow tools. As of March 1, 2010, the engineers at Darkroom and the people with PhotoReflect built an entirely new publisher that is completely owned and managed by PhotoReflect. Also as of this time, Web edition was discontinued. If you love Web edition, know you will love Core and Professional editions even more (and all of your libraries and data will transfer over without issue) if you choose to upgrade to those editions.

I downloaded a trial version of Core and/or Pro Edition but I can't do much.

The trial versions will let you import your images but will be scanned with an ExpressDigital watermark. Pre-installed images and catalogs will not have this watermark. The reason we did this was so you can see everything you can do and take your time, since there is no time limit to use these downloads.

In addition, you will not be able to connect your printer or camera drivers in evaluation mode. Nor will you be able to save any templates and borders you create in this mode. Beyond those limitations, you are able to enjoy the features of Darkroom in order for you to determine which product is right for you.

I can’t decide if I should get Core or Pro. Can you explain the differences in a couple of sentences?

Yes! Core and Pro are very similar, but Pro offers Networking capabilities (Pro acts as the Server while Core is used for Clients). Pro also offers direct printer drivers (Click here for a list). This makes printing faster and easier (especially to Mini Labs and Large Format Printers). Please note that Core will print to any printer with a Windows printer driver.
